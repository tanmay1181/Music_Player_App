# MyMusicApp(SanSic) 🎵

A beautiful Jetpack Compose music player built with ExoPlayer and Room.

## 🔥 Features
- Play, pause, and seek songs
- Save favorites
- Shuffle & loop modes
- Beautiful Now Playing screen
- Search Songs
- Create Playlists
